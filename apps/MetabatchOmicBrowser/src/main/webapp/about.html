Using MetaBatch Omic Browser

To view an analysis, click the "View Analysis" link under the View Results column or use the Previous or Next buttons at the top of the page.

To download data, click the "Data" link under the Data Download column or use the Data button in Data Viewer.

To download results for use with the standalone viewer, click the "Results" link under the Results Download column or use the Results button in Data Viewer.

To bookmark a URL for a search and diagram, click the "Copy URL" button at the top of the page. Paste the URL and bookmark that URL.
